# Triumph Guide

Интерактивный справочник для работы с Open WebUI. Современное минималистичное веб-приложение на Next.js с аутентификацией через Supabase.

## Технический стек

- **Frontend**: Next.js 14+ (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Backend/Auth**: Supabase
- **Icons**: Lucide React

## Возможности

- ✅ Аутентификация пользователей через Supabase
- ✅ Защищенные маршруты (middleware)
- ✅ Минималистичный дизайн в стиле Notion
- ✅ Адаптивная верстка (mobile-first)
- ✅ Навигация снизу (Bottom Navigation)
- ✅ Топ-бар с настройками
- ✅ Модальное окно настроек с функцией выхода
- ✅ Страницы: Модели, Промпты, Функции
- ✅ Детальные страницы для моделей и промптов

## Установка

### 1. Установка зависимостей

```bash
npm install
```

### 2. Настройка Supabase

1. Создайте проект на [supabase.com](https://supabase.com)
2. Включите Email Auth в настройках Authentication
3. Скопируйте Project URL и anon key

### 3. Переменные окружения

Создайте файл `.env.local` в корне проекта:

```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_project_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```

Замените `your_supabase_project_url` и `your_supabase_anon_key` на ваши значения из Supabase.

### 4. Создание тестового пользователя

В Supabase Dashboard:
1. Перейдите в Authentication > Users
2. Нажмите "Add user"
3. Создайте пользователя с email и паролем

### 5. Запуск приложения

```bash
npm run dev
```

Приложение будет доступно по адресу [http://localhost:3000](http://localhost:3000)

├── layout/

## Использование

### Вход в систему

1. Откройте приложение
2. Вы будете автоматически перенаправлены на `/login`
3. Введите email и пароль пользователя, созданного в Supabase
4. После успешного входа вы попадете на страницу "Модели"

### Навигация

Используйте нижнюю панель навигации для перехода между разделами:
- **Модели**: 5 карточек моделей (GPT5, Ассистент, Экспериментатор, Писатель, Художник)
- **Промпты**: 4 карточки промптов (Артефакты, Церемонии, Объекты, Схема)
- **Функции**: Пока в разработке

### Настройки и выход

1. Нажмите на иконку настроек в верхнем правом углу
2. В открывшемся модальном окне нажмите "Выйти из аккаунта"

## Дизайн

Приложение следует принципам минимализма в стиле Notion:

- **Цвета**: Белый фон (#FFFFFF), черный текст (#000000), серый для вторичного текста (#6B7280)
- **Типографика**: System font stack для оптимальной читаемости
- **Интерактивность**: Плавные transitions (200ms), hover-эффекты на карточках
- **Пространство**: Много белого пространства для комфортного восприятия

## Разработка

### Добавление нового раздела

1. Создайте страницу в `app/(protected)/your-section/page.tsx`
2. Добавьте маршрут в `BottomNav.tsx`
3. Импортируйте иконку из `lucide-react`

### Добавление контента

Контент для детальных страниц моделей и промптов можно добавить, отредактировав соответствующие файлы:
- `app/(protected)/models/[id]/page.tsx`
- `app/(protected)/prompts/[id]/page.tsx`
